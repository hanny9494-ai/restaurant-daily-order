# L0-L6 审批后台原型图（V2.5）

## 1) 信息架构
- 页面A：`/knowledge` 总览（待办、发布、告警）
- 页面B：`/knowledge/l0/queue` L0 审批队列
- 页面C：`/knowledge/upload` L1-L6 上载
- 页面D：`/knowledge/restart` 会话记忆与重启

## 6) 前端最小功能清单
- `L0队列`：列表、过滤、排序、冲突提示
- `L0审阅`：Approve / Reject / Need-Evidence / Publish
- `L1-L6上载`：单条与批量
- `AI辅助`：链接解析 -> 建议层级 -> 生成草稿JSON
- `重启能力`：读取 `restart_packet` 一键续跑
